# Yolo-Pyqt5

yolo with pyqt implement in pytorch.  

### untitled.ui -> pyqt5 ui文件  
### gui.py -> 由untitled.ui转换出来的py文件
### run_gui.py -> 运行gui主程序
### main.py -> 调试程序
### yolo.py -> yolo模块代码
### track_utils -> 存放跟踪的代码
### yolo_utils -> 存放关于yolo模型函数的代码

# 注意事项
1. 最好用最新版的去训练得到模型来到这个项目下进行使用,在网上的有些代码是版本比较旧的,不一定可以通用.
2. yolov5版本是基于v7.0.
